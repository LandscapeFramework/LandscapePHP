Getting Started With Landscape
===============================

This is a short tutorial (you will take ~20-30min) which will explain how to
create an Application using Landscape. It will be demonstrated by creating a
super simple 'note-pad-like' application on the server. All visitors can post
little entries which can be searched and read by others.
